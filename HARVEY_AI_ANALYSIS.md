# Детальный анализ ИИ-стартапа Harvey

## Общая информация о Harvey

**Harvey** — это специализированная платформа искусственного интеллекта, разработанная специально для юридических фирм, профессиональных сервисных компаний и корпораций из списка Fortune 500. Компания была основана в 2022 году и с тех пор привлекла значительные инвестиции:

- Ноябрь 2022: $5 млн
- Апрель 2023: $23 млн (оценка $715 млн)
- Декабрь 2023: $80 млн
- Июль 2024: $100 млн (оценка $1,5 млрд)
- Февраль 2025: $300 млн (оценка $3 млрд)
- Июнь 2025: $300 млн (оценка $5 млрд)
- Декабрь 2025: $160 млн (оценка $8 млрд)

Harvey активно сотрудничает с крупными юридическими фирмами, такими как Paul Weiss и Allen & Overy.

## 1. VAULT - Защищенное хранилище документов

### Основное назначение
Vault представляет собой защищенное рабочее пространство для проектов, где пользователи могут загружать, хранить и анализировать большие объемы документов с помощью генеративного ИИ.

### Ключевые возможности

#### 1.1 Внутренние знания (Internal Knowledge)
- **Мгновенный доступ** к знаниям организации без необходимости ручной загрузки или поиска документов
- Автоматическая индексация и организация документов
- Интеллектуальный поиск по содержимому документов

#### 1.2 Качественная аналитика
- **97% точность** при извлечении ключевых терминов из различных типов документов
- Поддержка анализа различных форматов документов (соглашения, судебные материалы, клиентские файлы)
- Высокая степень достоверности при извлечении информации

#### 1.3 Масштабируемость
- Поддержка хранения **более 100 000 документов** в одном хранилище
- Эффективная обработка больших объемов данных
- Оптимизированная производительность при работе с крупными наборами документов

#### 1.4 Совместное использование Vault
- Простое совместное использование хранилищ между пользователями
- Детальный контроль доступа (просмотр, редактирование)
- Аудит действий для соблюдения требований безопасности
- Совместная работа над проектами

#### 1.5 Интеграции
- **SharePoint интеграция**: импорт и синхронизация файлов с популярными системами управления документами
- Бесшовная интеграция с другими компонентами платформы Harvey
- API для интеграции с внешними системами

#### 1.6 Безопасность
- Защищенное хранение конфиденциальных данных
- Соответствие стандартам безопасности юридических организаций
- Детальные настройки доступа и аудит действий
- Корпоративная аутентификация

### Технические особенности
- Использование генеративного ИИ для анализа документов
- Высокая точность извлечения информации (97%)
- Масштабируемая архитектура для больших объемов данных
- Интеграция с корпоративными системами

## 2. ASSISTANT - Персональный ИИ-помощник

### Основное назначение
Assistant — это персональный помощник, адаптированный к специфике работы пользователя, который позволяет делегировать сложные задачи на естественном языке.

### Ключевые возможности

#### 2.1 Контекстно-зависимые запросы
- Выполнение безопасных запросов с учетом контекста
- Использование внутренних источников фирмы (веб-ресурсы, Vault)
- Интеграция с корпоративными базами знаний
- Точные ответы с соответствующими ссылками на источники

#### 2.2 Голосовой ввод (Диктовка)
- Возможность голосового ввода запросов
- Работа без использования рук (hands-free)
- Поддержка мобильных устройств
- Быстрое и удобное взаимодействие

#### 2.3 Magic Prompt
- Функция "Magic Prompt" для улучшения формулировки запросов
- Автоматическое уточнение запросов в один клик
- Получение более точных и релевантных результатов
- Оптимизация взаимодействия с ИИ

#### 2.4 Поиск в Vault
- Доступ к соглашениям, документам и клиентским файлам
- Прикрепление документов к запросам
- Анализ документов с помощью ИИ
- Интеграция с хранилищем документов

#### 2.5 Сканирование и загрузка
- Фотографирование контрактов или приложений
- Сохранение документов в Vault
- Прикрепление документов к запросам
- Работа с мобильных устройств

#### 2.6 Аудио в текст
- Запись разговоров и встреч
- Генерация транскриптов с соответствующими дисклеймерами
- Автоматическая обработка аудио
- Интеграция с рабочими процессами

#### 2.7 История и синхронизация
- Просмотр прошлых запросов и потоков
- Синхронизация в реальном времени с веб-учетной записью
- Доступ к истории работы
- Непрерывность работы между устройствами

#### 2.8 Совместное использование
- Бесшовное совместное использование запросов
- Совместная работа над проектами Vault
- Обмен результатами с коллегами
- Коллаборативные функции

#### 2.9 Безопасный вход
- Соответствие стандартам безопасности фирмы
- Корпоративная аутентификация
- Защита конфиденциальных данных
- Интеграция с корпоративными системами безопасности

### Функциональные области применения
- Юридические исследования
- Составление документов
- Анализ документов
- Ответы на вопросы по делам
- Подготовка отчетов
- Регуляторные вопросы
- Налоговые вопросы

### Мобильное приложение
- Полнофункциональное мобильное приложение
- Безопасная работа на ходу
- Доступ к делам и документам
- Все функции Assistant доступны в мобильной версии

## 7. Выводы и ключевые инсайты

### 7.1 Архитектурные решения
- **Модульная архитектура**: разделение на Vault, Assistant и Agents позволяет независимо развивать каждый компонент
- **Интеграция компонентов**: тесная интеграция обеспечивает бесшовный пользовательский опыт
- **Масштабируемость**: поддержка больших объемов данных и пользователей

### 7.2 UX/UI подход
- **Фокус на пользователе**: интуитивные интерфейсы с минимальной кривой обучения
- **Современные технологии**: использование React, TypeScript и TailwindCSS для создания высокопроизводительных интерфейсов
- **Интеграция с экосистемой**: работа в привычных инструментах (Microsoft Word, Outlook)

### 7.3 Функциональность
- **Vault**: мощное хранилище с высокой точностью извлечения информации (97%)
- **Assistant**: универсальный помощник с голосовым вводом и контекстным пониманием
- **Agents**: автоматизация сложных многоэтапных процессов с использованием продвинутых моделей рассуждений

### 7.4 Безопасность
- **Корпоративный уровень**: соответствие стандартам безопасности юридических организаций
- **Контроль доступа**: детальные настройки прав доступа и аудит действий
- **Защита данных**: безопасное хранение конфиденциальной информации

### 7.5 Рекомендации для изучения
1. **Vault**: изучить архитектуру хранения и индексации документов, методы достижения 97% точности
2. **Assistant**: проанализировать интеграцию с различными источниками данных, механизмы контекстного понимания
3. **Agents**: изучить модели рассуждений и планирования, подходы к автоматизации сложных процессов
4. **UX/UI**: проанализировать паттерны взаимодействия, компонентную архитектуру, подходы к производительности
