# Руководство по использованию Bugbot в Cursor

## Как активировать Bugbot

### Способ 1: Command Palette (рекомендуется)
1. Нажмите `Cmd+Shift+P` (Mac) или `Ctrl+Shift+P` (Windows/Linux)
2. Введите `Bugbot` или `Cursor: Run Bugbot`
3. Выберите команду для запуска сканирования проекта

### Способ 2: Через статус-бар
- Посмотрите в нижний правый угол окна Cursor
- Если видите индикатор/кнопку Bugbot — нажмите на неё

### Способ 3: Автоматический режим
Bugbot может работать автоматически:
- При сохранении файлов (`onSave`)
- При открытии проекта
- По расписанию

Для настройки автоматического режима:
1. Откройте настройки: `Cmd+,` (Mac) или `Ctrl+,` (Windows/Linux)
2. Найдите настройки Bugbot
3. Включите нужные опции

### Способ 4: Проверка конкретного файла
1. Откройте файл, который хотите проверить
2. Правый клик по файлу → "Run Bugbot on this file"
   или
3. Command Palette → "Bugbot: Scan Current File"

## Что делает Bugbot

Bugbot автоматически сканирует ваш код на наличие:
- Потенциальных багов
- Ошибок логики
- Проблем безопасности
- Антипаттернов
- Проблем производительности

## Результаты сканирования

После запуска Bugbot:
- Результаты отображаются в панели Problems (Проблемы)
- Или в отдельной панели Bugbot
- Ошибки будут подсвечены в коде

## Настройка Bugbot

### ⚠️ Важно: Изменение режима работы

Если вы видите настройку **"Only Run When Mentioned"**, это означает, что Bugbot работает **только при комментариях в Pull Request** (`bugbot run` или `@cursor review`).

Для локальной работы нужно изменить эту настройку:

1. Откройте настройки Bugbot в Cursor
2. Найдите опцию **"Only Run When Mentioned"**
3. **Выключите** эту опцию (или измените на "Always Run" / "Run on Save" / "Run on Demand")
4. Сохраните настройки

### Альтернатива: Ручной запуск

Если хотите оставить настройку "Only Run When Mentioned" для PR, но нужен локальный запуск:

**Вариант 1: Command Palette**
- `Cmd+Shift+P` (Mac) / `Ctrl+Shift+P` (Windows/Linux)
- Введите: `bugbot run` или `@cursor review`
- Нажмите Enter

**Вариант 2: Проверка через Chat**
- Откройте чат в Cursor (`Cmd+L` или `Ctrl+L`)
- Напишите: `@cursor review` или `bugbot run`
- Cursor запустит анализ кода

### Если Bugbot не запускается

Проверьте:
1. Убедитесь, что вы используете актуальную версию Cursor
2. Проверьте настройки расширений Cursor
3. Перезапустите Cursor IDE
4. Убедитесь, что настройка "Only Run When Mentioned" выключена для локальной работы
